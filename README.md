## What it does
Our project was developed to help people recycle, upcycle, and resell objects that would otherwise have gone to landfills and polluted the environment.
It is an app that lets you upload an image or take a picture of an object, which it then tells you if it is recycable, resellable, or able to be made into 
something useful and then gives additional information if needed.

## How we built it
We built it using python on Pycharm and Ollama for the visual language model to analyze the images and provide responses.

## Accomplishments that we're proud of
We integrated Google Maps API into the project.
